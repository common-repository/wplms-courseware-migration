WPLMS Courseware Migrate helps users to migrate all Courseware courses to WPLMS courses

== Description ==
WPLMS Courseware Migrate plugin is a free plugin developed to help users to migrate all of their Courseware courses to WPLMS courses. The users do not have to worry a lot and perform various steps to migrate their courses. All this is done by a simple click. With the courses all of the quizzes and the units will be migrated successfully with just one click.

Tutorial On how to setup and get started : <a href="http://vibethemes.com/documentation/wplms/knowledge-base/wp-courseware-wplms-migrate-plugin/">link</a>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS Courseware Migrate'
3. Activate WPLMS Courseware Migrate from your Plugins page. 

= From WordPress.org =

1. Download WPLMS Courseware Migrate.
2. Upload the 'wplms_courseware_migrate' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS Courseware Migrate from your Plugins page.


== Frequently Asked Questions ==

= Do I need the WPLMS theme and Courseware plugin both to begin? =

Yes, You need both the Courseware plugin and the WPLMS theme activated to begin the migration process

= Can I revert changes if I don't like it after migrating? =

No, right now you cannot revert the changes once the courses are migrated.
